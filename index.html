THE PUZZLE

üß© THE PROBLEM STATEMENT

What are we solving?

The Challenge: In many situations (hospitals, warehouses, or home shopping), people carry heavy loads and cannot use their hands to steer a robot.

The Solution: A hands-free, autonomous vehicle that detects the user and follows them automatically without any remote control or human touch.

Hypothesis: By using Ultrasonic Sonar, we can calculate distance in real-time to maintain a "Safety Zone" behind the user.

SYSTEM OVERVIEW

üåç EXPLORING THE SOLUTION

Project Overview

The Goosy Gang Follower is an intelligent machine. It works using three main sections:

  1. The Input: Ultrasonic sensor (The Eyes).
  2. The Processor: Arduino Uno (The Brain).
  3. The Output: L298N Motor Driver and DC Motors (The Legs).

This creates a "Feedback Loop" where the robot is constantly checking if it is too close or too far.
